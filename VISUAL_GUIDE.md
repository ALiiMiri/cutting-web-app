# راهنمای تصویری دسترسی به پنل مدیریت کاربران

## مرحله 1️⃣: باز کردن صفحه اصلی
در مرورگر به این آدرس بروید:
```
http://127.0.0.1:5000
```

## مرحله 2️⃣: وارد شدن به سیستم

شما باید صفحه ورود را ببینید که شامل:
- یک فرم با زمینه "نام کاربری"
- یک فرم با زمینه "رمز عبور"
- دکمه "ورود"

### اطلاعات ورود:
```
نام کاربری: admin
رمز عبور: admin123
```

## مرحله 3️⃣: مشاهده صفحه اصلی

بعد از ورود موفق، باید این موارد را ببینید:

### 🎨 نوار منوی بنفش در بالای صفحه:
```
┌─────────────────────────────────────────────────────────────┐
│  👤 admin  🔑 مدیر    [👥 مدیریت کاربران] [🔐 تغییر رمز] [🚪 خروج]  │
└─────────────────────────────────────────────────────────────┘
```

این نوار شامل:
- سمت راست: نام شما (admin) و نقش شما (مدیر)
- سمت چپ: سه دکمه
  - 👥 **مدیریت کاربران** ← روی این کلیک کنید!
  - 🔐 تغییر رمز
  - 🚪 خروج

## مرحله 4️⃣: ورود به پنل مدیریت کاربران

روی دکمه **"👥 مدیریت کاربران"** کلیک کنید.

یا مستقیماً به این آدرس بروید:
```
http://127.0.0.1:5000/admin/users
```

## مرحله 5️⃣: پنل مدیریت کاربران

در این صفحه می‌بینید:

### بخش 1: فرم ایجاد کاربر جدید
```
┌─── ایجاد کاربر جدید ────────────────────────┐
│ نام کاربری: [______]                        │
│ رمز عبور موقت: [______]                     │
│ نقش: [کارمند ▼]                             │
│ [ایجاد کاربر]                                │
└──────────────────────────────────────────────┘
```

### بخش 2: جدول لیست کاربران
```
┌─────────────────────────────────────────────────────────────────┐
│ شناسه │ نام کاربری │ نقش │ وضعیت │ تاریخ ایجاد │ عملیات        │
├───────┼────────────┼─────┼───────┼────────────┼──────────────┤
│   1   │   admin    │ مدیر │ فعال  │ 2025-...   │ [تغییر نقش]  │
│       │            │     │       │            │ [فعال/غیرفعال]│
│       │            │     │       │            │ [ریست رمز]   │
└───────┴────────────┴─────┴───────┴────────────┴──────────────┘
```

## 🎯 عملیات‌های موجود:

### 1. ایجاد کاربر جدید
- نام کاربری را وارد کنید
- یک رمز موقت بگذارید (کاربر در اولین ورود باید تغییرش دهد)
- نقش را انتخاب کنید: مدیر / کارمند / فقط مشاهده
- روی "ایجاد کاربر" کلیک کنید

### 2. تغییر نقش کاربر
- روی دکمه "تغییر نقش" کلیک کنید
- یک پنجره popup باز می‌شود
- نقش جدید را انتخاب کنید
- تایید کنید

### 3. فعال/غیرفعال کردن کاربر
- روی دکمه "فعال/غیرفعال" کلیک کنید
- کاربر فوراً فعال یا غیرفعال می‌شود

### 4. ریست رمز عبور
- روی "ریست رمز" کلیک کنید
- رمز جدید را وارد کنید
- کاربر در ورود بعدی مجبور به تغییر رمز می‌شود

---

## ⚠️ اگر صفحه را نمی‌بینید:

### علت 1: لاگین نکرده‌اید
راه حل: به صفحه اصلی بروید و وارد شوید

### علت 2: با کاربر غیر admin وارد شده‌اید
راه حل: خارج شوید و با admin وارد شوید

### علت 3: منوی بالا را نمی‌بینید
راه حل: صفحه را رفرش کنید (F5)

---

## 📝 نکات مهم:

1. ✅ فقط کاربر admin می‌تواند پنل مدیریت کاربران را ببیند
2. ✅ رمزهای موقت باید حداقل 6 کاراکتر باشند
3. ✅ کاربران جدید در اولین ورود مجبور به تغییر رمز هستند
4. ✅ نمی‌توانید خودتان را غیرفعال کنید یا نقش خودتان را تغییر دهید
5. ✅ کاربران غیرفعال نمی‌توانند وارد سیستم شوند

---

## 🆘 کمک سریع:

**مشکل: دکمه "مدیریت کاربران" را نمی‌بینم**
- بررسی کنید با `admin` وارد شده باشید
- صفحه را رفرش کنید
- اگر باز هم نیست، خارج شوید و دوباره وارد شوید

**مشکل: خطای 403 (Forbidden) می‌گیرم**
- شما با کاربر غیر admin وارد شده‌اید
- خارج شوید و با admin وارد شوید

**مشکل: صفحه ورود بارها باز می‌شود**
- Cookie‌های مرورگر را پاک کنید
- مرورگر را ببندید و دوباره باز کنید
- با admin وارد شوید
